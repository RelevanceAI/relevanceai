Client
========================================

You can instantiate a client with Relevance AI by running:

.. code-block::

    from relevanceai import Client
    client = Client()

If you have a token already, you can run this: 

.. code-block::

    from relevanceai import Client
    client = Client(token="...")

If you are missing a token, you can get your token [here](https://cloud.relevance.ai/sdk/api).
